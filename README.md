
# 📊 Excel Sales Performance Analysis Dashboard

## 📌 Project Overview

The **Sales Performance Analysis Dashboard** is an advanced Excel-based project designed to analyze and visualize sales data efficiently. It transforms raw data into meaningful insights using interactive dashboards, charts, and formulas.

This project helps in tracking business performance, identifying trends, and supporting data-driven decision-making.

---

## 🎯 Aim

To design and develop an interactive Excel dashboard that analyzes sales data and provides clear insights into performance metrics.

---

## 🎯 Objectives

* To organize and manage sales data effectively
* To calculate key metrics like total sales, profit, and growth
* To visualize data using charts and dashboards
* To identify top-performing products and regions
* To simplify decision-making using data insights

---

## 📂 Features

* 📈 Interactive Dashboard
* 📊 Charts (Bar, Line, Pie, etc.)
* 🔍 Data Filtering (Slicers)
* 🧮 Excel Formulas (SUM, IF, VLOOKUP, etc.)
* 📅 Time-based Analysis
* 💰 Profit & Loss Tracking

---

## 🛠️ Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Excel Functions
* Conditional Formatting

---

## 📊 Key Metrics Included

* Total Sales
* Total Profit
* Sales by Region
* Sales by Product
* Monthly Sales Trends
* Profit Margin

---

## 📈 How to Use

1. Open the Excel file
2. Navigate to the Dashboard sheet
3. Use slicers/filters to analyze data
4. Review charts and insights
5. Modify data to see dynamic updates

---

## 📌 Formulas Used

* **SUM** – To calculate totals
* **AVERAGE** – To find mean values
* **IF** – For conditional logic
* **VLOOKUP/XLOOKUP** – For data retrieval
* **MAX/MIN** – To find highest & lowest values
* **IFERROR** – To handle errors
